# Alert 提示框使用示例

本页面演示了如何使用GitHub风格的Alert语法创建各种类型的提示框。

## 基本语法

使用以下shortcode语法创建提示框：

```markdown
{{< alert "type" >}}
这里是提示框的内容
{{< /alert >}}
```

或者带自定义标题：

```markdown
{{< alert "type" "自定义标题" >}}
这里是提示框的内容
{{< /alert >}}
```

创建可关闭的提示框：

```markdown
{{< alert "type" "标题" true >}}
这里是可关闭的提示框内容
{{< /alert >}}
```

创建可折叠的提示框（不可关闭）：

```markdown
{{< alert "type" "标题" false true >}}
这里是只能折叠的提示框内容
{{< /alert >}}
```

创建既可关闭又可折叠的提示框：

```markdown
{{< alert "type" "标题" true true >}}
这里是既可关闭又可折叠的提示框内容
{{< /alert >}}
```

## 使用建议

- **TIP**: 用于提供有用的技巧和建议
- **WARNING**: 用于重要的注意事项和潜在风险
- **ERROR**: 用于错误信息和严重问题
- **INFO**: 用于补充信息和背景说明  
- **NOTE**: 用于一般性的备注和说明

选择合适的类型可以让你的文档更加清晰和易读！

{{< alert "info" "详细说明" false true >}}
这是一个可折叠的提示框。点击标题栏或右侧的箭头按钮可以折叠/展开内容。

### 折叠功能的优势

- **节省空间** - 在页面上显示更多信息而不占用太多空间
- **渐进式披露** - 用户可以选择是否查看详细内容
- **更好的用户体验** - 避免信息过载

### 使用场景

1. **详细的技术说明**
2. **可选的补充信息**
3. **FAQ 问答**
4. **长篇的注意事项**

你可以点击标题栏的任意位置来切换折叠状态，也可以使用键盘的 **Enter** 或 **空格键**。
{{< /alert >}}

**注意事项：**

- SSL 证书需要在到期前更新

### 如何使用这个功能？

这个提示框只能折叠和展开，但不能被关闭。这种设计适用于：

1. **重要的永久性信息** - 用户需要随时查看但可以暂时隐藏
2. **帮助文档** - 提供详细说明但不占用太多空间
3. **配置选项** - 高级设置可以折叠起来

### 语法说明

```markdown
{{< alert "type" "标题" false true >}}
内容...
{{< /alert >}}
```

参数说明：
- 第1个参数：提示框类型（tip, warning, error, info, note）
- 第2个参数：自定义标题
- 第3个参数：`false` = 不可关闭
- 第4个参数：`true` = 可折叠
{{< /alert >}}
